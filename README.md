| US008       | Actualización de Perfil | Como usuario, quiero poder actualizar mi información personal y de contacto en mi perfil de usuario. | - |

| US013       | Gestión de Prescripciones | Como médico, quiero poder prescribir medicamentos de manera electrónica y enviar las recetas a los pacientes de forma segura. | - |
| US014       | Gestión de Agenda de Médicos | Como médico, quiero poder gestionar mi agenda de citas y disponibilidad en la plataforma. | - |
| US015       | Gestión de Perfil de Médico | Como médico, quiero poder actualizar mi perfil profesional y de contacto en la plataforma. | - |
| US016       | Alertas de Seguridad | Como administrador, quiero recibir alertas automáticas sobre posibles problemas de seguridad en la plataforma para tomar medidas inmediatas. | - |
| US017       | Informes Estadísticos | Como administrador, quiero poder generar informes estadísticos sobre el uso de la plataforma y la atención médica proporcionada. | - |
| US018       | Gestión de Roles y Permisos | Como administrador, quiero poder asignar roles y permisos a los usuarios y médicos en la plataforma para garantizar la seguridad y la privacidad de los datos. | - |
| US019       | Integración de Sistemas de Pago | Como administrador, quiero integrar sistemas de pago seguros en la plataforma para facilitar transacciones entre pacientes y médicos. | - |
| US020       | Gestión de Reportes de Usuarios | Como administrador, quiero poder recibir y gestionar reportes de usuarios sobre problemas técnicos o de comportamiento en la plataforma. | - |
